FrameDock API Documentation

Simple reference for the available endpoints.

/lookup/<word>

Returns dictionary meanings for a word.

Method: GET

Example:

GET /lookup/test

Response:

{
  "word": "test",
  "meanings": ["..."]
}

Errors:

{
  "error": "not found"
}

/ai/question/<query>

Asks the AI model a question.

Method: GET

Example:

GET /ai/question/what%20is%20linux

Response:

{
  "answer": "..."
}
